## 数组查找
### 顺序查找
* 有序、无序线性表均可使用
* 从线性表一端开始，逐个检查元素，直到找到目标元素或查找完所有元素。
```C
 int SequenceSearch(A, value){
	 for i = 0, i<n,i++{
		 if A[i] == value return i;
	 }
	 return -1; //未找到。
 }

// 时间复杂度 O(n)

```
### 折半查找
* 只有有序列表可以用。
* 将表分成两段，并确定元素在哪一半中，然后对选择那一半重复此过程，直到找到目标元素或查找完所有元素。
```c
int BinarySearch(A,value){
	low = 0, high = n-1; //初始化两端
	while(low<=hign){    //结束条件：low比 hign 高，或 hign 比 low 低。
		mid = (low+high)/2; //确定中间位置
		if A[mid] == value{
			return mid;
		}else if(A[mid]<value){ // 目标元素在右侧
			low = mid + 1;
		}else{//目标元素在左侧
			hign = mid - 1;
		}
	}
	return -1; //未找到
}
// 时间复杂度 O(log n)
```
### 分块查找
* 一种特殊情况，块内无序，块间有序。
* 先对索引表进行顺序或者二分，确定元素所在的块，然后在块中进行顺序查找。
