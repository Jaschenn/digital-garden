### 常用结论
* n个顶点的树中有n-1条边。
* 树的结点数n等于所有节点的度+1.
### 存储结构
#### 双亲表示法

双亲表示法主要是使用一个数组，其中每个结点都有一个指示其双亲结点在数组中位置的索引

```c
#define MAXSIZE 100
typedef struct {
    int data;         // 结点数据
    int parent;       // 双亲的位置
} PTNode;

typedef struct {
    PTNode nodes[MAXSIZE];  // 结点数组
    int n;                  // 结点数
} PTree;
```

#### 孩子表示法

孩子表示法将每个结点的孩子结点排列起来，以单链表作为存储结构。然后再用一个数组与之相配合。

```c
#define MAXSIZE 100

// 孩子结点
typedef struct ChildNode {
    int child;                   // 孩子结点在数组中的位置
    struct ChildNode* next;     // 下一个孩子
} *ChildPtr;

// 表头结构
typedef struct {
    int data;                   // 结点数据
    ChildPtr firstchild;        // 第一个孩子的指针
} CTBox;

typedef struct {
    CTBox nodes[MAXSIZE];       // 结点数组
    int n;                      // 结点数
} CTree;
```

#### 孩子兄弟表示法

孩子兄弟表示法是将树转化为二叉树的形式来存储。每个结点有两个指针，一个指向它的第一个孩子，另一个指向它的右兄弟。

```c
typedef struct CSNode {
    int data;                     // 结点数据
    struct CSNode* firstchild;   // 第一个孩子
    struct CSNode* rightsib;     // 右兄弟
} CSNode, *CSTree;
```

#### 树、二叉树、森林相互转换
* 树转成二叉树：左孩子右兄弟表示法
* 森林转成二叉树：将每一棵树转成二叉树，然后第一棵树的右孩子分别是其他的树根。
