## 图的存储

| 方法    | 特点                               |
| ----- | -------------------------------- |
| 邻接矩阵  | 适用于有向图、无向图                       |
| 邻接表   | 适用于有向图、无向图，但是无向图中会重复存储，因此引入邻接多重表 |
| 邻接多重表 | 只适用于无向图，减少重复存储。                  |
| 十字链表  | 只用于存储有向图，可用于高效遍历。                |


### 邻接矩阵
特点：无向图的邻接矩阵一定是个对称阵，且主对角元素都是 0.


```c
// 邻接矩阵存储的图 求入度和出度
// 遍历第 i 行，有多少个不为 0 的元素即为节点 i 的出度
// 遍历第 j 列，有多少个不为 0 的元素，即为节点 j 的入度
int get_in_degree(int[][] g, int num){
	// g 是临界矩阵，num 是总节点个数

	// 统计第 i 个节点的出度
	int result = 0;
	for(int k = 0, k<num,k++){
		if(g[i][k]!=0){
			result++;
		}
	}
	print(result); // 遍历那一行即可得到节点 i 的出度。
}

```
